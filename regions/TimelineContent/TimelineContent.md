# Node: `TimelineContent`

> **Region**: [TimelineContent](_REGION.md)
> **Exists in**: Dialog timeline files only (not standalone)
> **Note**: Embedded region — no standalone data file in standard UnpackedData.

---

## Status

**Dialog-embedded region.** `TimelineContent` appears as a child within dialog
timeline files, containing the actual timeline data for a dialog sequence.
This includes animation channels, camera keys, effect timing, and audio cues.

This is a deeply nested structure within each dialog's timeline definition.

## Vanilla Scope

Not countable — embedded per-dialog across thousands of dialog files.
